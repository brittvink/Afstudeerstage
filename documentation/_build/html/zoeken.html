zoeken module

File: zoeken.py Created: 2022/02/17 Last Changed: 2022/02/22 Author: B.Vink

This pythonscript is used to read a database table and find all results that match input filters

The filters are given with the input argument (-s). The database is readed and results are printed in terminal
